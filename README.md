# CineVerse AI 🎬

CineVerse AI is an AI-powered cinema discovery platform where users can explore actor and actress filmographies, hear interactive star-style greetings, ask cinema-related questions, and get mood-based movie or series recommendations.

## Features Planned

- Actor and actress profile explorer
- Career-wise movie timeline
- Movie and series recommendation system
- AI cinema chatbot
- Text-to-speech star greeting
- Movie analytics dashboard
- Streaming availability integration
- Full-stack deployment

## Tech Stack

- Frontend: React + TypeScript
- Backend: Python + FastAPI
- Database: PostgreSQL
- Movie Data: TMDB API
- AI: Generative AI API
- Recommendation System: Python + scikit-learn
- Voice: Web Speech API / Cloud TTS
- Deployment: Docker
